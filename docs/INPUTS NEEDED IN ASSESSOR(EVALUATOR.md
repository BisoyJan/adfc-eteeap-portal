Inputs Needed in Assessor (Evaluator) Account

### Communication
\-Inbox for receiving learner messages  (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Message notifications (email or in-app) (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Send messages to individual learners (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Send bulk messages to multiple learners (not available in either Admin or Assessor accounts)

\-Message templates for common responses (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Sent messages folder (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Attach files to messages (currently seen only on the Admin side — this must also be available in the Assessor account)

### Dashboard
\-Learners with incomplete requirements (currently seen only on the Admin side — this must also be available in the Assessor account)

\-Active learners vs. completed learners (currently seen only on the Admin side — this must also be available in the Assessor account)
