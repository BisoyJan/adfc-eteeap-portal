Inputs Needed in Applicant/Learner Account

### Dashboard
\-Upcoming deadlines or requirements (if this exists in the assessor account, it must also appear in the learner’s dashboard)

\-Announcements from the ETEEAP program (there must be Announcements in the Admin sidebar; in the applicant’s account, they should also see announcements relevant to them)

### Communication Features
\-Read receipts (ability to know if a message was seen)

### Report and Document Generation Features (in the Sidebar)
\-My complete portfolio summary

\-My competency achievement report

\-My assessment results summary

\-Course waiver recommendations report

\-Progress-to-date report
